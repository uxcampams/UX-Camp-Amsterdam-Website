## GENERAL RULES
- No frameworks — plain HTML, CSS, vanilla JS only
- All pages share the same navbar and footer (copy-paste the HTML or use JS includes)
- All headings: ALL CAPS, bold, letter-spacing 0.05em
- All buttons: sharp corners (border-radius: 0), never rounded
- Hover on red buttons: darken to #8B0000
- Responsive at 768px breakpoint
- Clean, minimal — no unnecessary decorative elements
- Add HTML comments throughout so a beginner can follow the structure
