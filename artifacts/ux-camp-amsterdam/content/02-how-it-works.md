# How It Works

**Source URL:** https://uxcampams.com/how-it-works

---

## What is UXcamp Amsterdam?

UXcamp Amsterdam is a **one-day, non-profit event** that brings together a community of UXers from Amsterdam and beyond for a day of knowledge sharing, learning, and discussions on User Experience.

---

## The BarCamp Format

The event operates on a **BarCamp model** — participants create the agenda on the day itself.

All participants are offered a chance to give a talk, run a discussion, or a workshop lasting **25 minutes** (extendable for workshops).

### The Madness Session
The morning kicks off with a **"Madness Session"** where:
1. Speakers pitch their topics in **30 seconds**
2. The audience votes by raising hands
3. The schedule for the day is set

---

## Session Types

| Format | Description |
|--------|-------------|
| **Presentations** | Overview of a topic in the UX field |
| **Case Studies** | Applied UX knowledge from real projects |
| **Workshops** | Hands-on skill development |
| **Structured Discussions** | Focused conversations for shared learning |

---

## Participation Philosophy

> **NO SPECTATORS, ONLY PARTICIPANTS.**

The event emphasises **involvement, inclusion, community and learning**.

Even non-presenters contribute through:
- Active session engagement
- Asking questions
- Sharing insights
- Assisting with event logistics (registration, setup)

---

## Event Structure

- **Before lunch (12:30):** 3 session rounds
- **After lunch:** 6 session rounds
- **End of day:** Closing remarks & after-party
